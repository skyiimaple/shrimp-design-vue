# Image 图片预览

`Image` 组件是基于 `viewerjs` 的二次开发，提供了更加便捷的图片预览功能，并内置了图片加载错误的占位处理。

## 基本用法

<preview path="../demos/img/basic.vue"></preview>

## Props

| 参数       | 说明                       | 类型      | 默认值  |
| ---------- | -------------------------- | --------- | ------- |
| `src`      | 图片资源地址               | `string`  | `''`    |
| `preview`  | 是否开启预览功能           | `boolean` | `false` |
| `errorSrc` | 图片加载失败时的兜底图地址 | `string`  | `''`    |
| `alt`      | 图片描述                   | `string`  | `''`    |

## Slots

| 插槽名    | 说明               | 参数                      |
| --------- | ------------------ | ------------------------- |
| `default` | 自定义图片渲染逻辑 | `{ show, viewer, error }` |

## Events

| 事件名  | 说明               | 参数               |
| ------- | ------------------ | ------------------ |
| `error` | 图片加载失败时触发 | `(payload: Event)` |
